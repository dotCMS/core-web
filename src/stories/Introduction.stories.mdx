# DotCMS Components Library

This project contains a list of the components used to build DotCMS UI.

## PrimeNG

PrimeNG is a rich set of open source native Angular UI components. We use as lego pieces to build our UI, including custom components.

## DotCMS Components

Using PrimeNG components we have created a library of custom components that we reuse along our app. This components are in continuous change to adjust to new DotCMS feaures.
